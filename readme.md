当前目录放了两个商品，里面存放了商品图片和资料，资料的标题是商品名，资料最后一行是亚马逊链接

› 如何增加更多商品


• 现在新增商品需要做两件事：放文件夹 + 在 script.js 里登记。

  步骤

  - 新建文件夹：比如 3/
  - 放入：商品图片 + doc.md（第一行是标题，最后一行是亚马逊链接）
  - 然后编辑 script.js，在 products 数组里追加一项

  示例（追加一项）：

  {
    id: "3",
    accent: "#5a86f7",
    badge: "2020 系列 · 角码套装",
    images: [
      "xxx.jpg",
      "yyy.jpg"
    ]
  }
